Adjust controls above to visualize agent coordination patterns

Reading the timeline: Each row shows one agent's behavior across interaction rounds (left to right). Each block represents activity intensity in that round.

Color hue = agent identity (starts similar, differentiates over time as coordination emerges). Brightness = activity level in that round. Pattern = coordination type:

  • Similar colors throughout → Control: minimal differentiation, agents remain independent
  • Colors diverge gradually → Personas: stable role identities emerge over time
  • Colors diverge rapidly + complementary brightness patterns → Personas+ToM: agents actively coordinate (when one is bright, others dim = complementarity)

TDMI components

Redundant --

Information shared by all agents (everyone reacting to the same signal)

Unique --

Agent-specific patterns (each agent's individual style or role)

Synergistic --

Cross-agent coordination (patterns only visible when observing the group)

Key: Synergy is the signal for genuine coordination—if it's near zero, agents aren't truly coordinating despite appearing to work together.

Group success rate on the guessing task. High coordination should correlate with better outcomes.
