🧠 Key idea

Zero-shot learning lets an LLM perform a task without task-specific examples by pairing pretrained knowledge with a natural-language prompt that names the task and the label words (verbalizer).

Formal view: choose the label with highest conditional likelihood
$$\hat y = \arg\max_y P(y \mid x, p),\; P(y \mid x, p) = \operatorname{softmax}_y\!\left(\frac{s_y(x,p)}{T}\right)$$
🎯 Zero-shot
  • No examples in prompt
  • Only label words supplied
  • Fast, brittle if labels ambiguous
🏹 Zero-shot + instruction
  • Adds explicit instruction
  • Improves label calibration
  • Cheap quality boost
🧪 Few-shot (k-shot)
  • Provides k labelled examples
  • Shows format and boundary
  • Higher token & curation cost

Mechanism: prompt tokens condition internal representations; label tokens compete through next-token probabilities. Instructions bias distribution; few-shot examples steer via pattern completion.
